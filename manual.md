Дальнейшие инструкции:

Откройте папку проекта:

при открытии новой командной строки:
cd Desktop\Sunny_Kids

затем создайте виртуальное окружение с помощью команды:
python -m venv venv

и активируйте:
venv\Scripts\activate

2. Установите все инструменты из файла requirements.txt:
pip install -r requirements.txt


3. Откройте и изучите проект
