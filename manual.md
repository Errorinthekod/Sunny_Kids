Sunny Kids
Инструкция по установке проекта:

0. Откройте командную строку.

1. Откройте Рабочий Стол с помощью команды:
```cd Desktop```

2. Склонируйте репозиторий проекта:
```git clone https://github.com/Errorinthekod/Sunny_Kids```

3. Откройте папку проекта:
```cd Sunny_Kids```

4. затем создайте виртуальное окружение с помощью команды:
```python -m venv venv```

5. активируйте:
```venv\Scripts\activate```

6. Установите все инструменты из файла requirements.txt:
```pip install -r requirements.txt```

7. Выведите текущую ветвь:
```git branch```
8. При выходе ветви main, создайте и перейдите на свою командой:
```git checkout -b askhat```
*вместо askhat пишите свое имя*

7. Откройте и изучите проект
